What are Attribute Selectors?

Attribute selectors allow you to style HTML elements based on their attributes and attribute values. Think of them as special filters that help you target specific elements!

Imagine you're organizing a library:

  • All books = All elements
  • Books with red covers = Elements with specific attributes
  • Books by a specific author = Elements with specific attribute values

Basic Attribute Selectors

1. [attribute]

Selects elements that have the specified attribute

[title]

This has a title

This doesn't have a title

2. [attribute="value"]

Selects elements with exact attribute value

[type="button"]

3. [attribute~="value"]

Selects elements with attribute containing the word

[class~="highlight"]
Highlighted Important highlight Normal text

4. [attribute|="value"]

Selects elements with attribute starting with value

[lang|="en"]

English

English US

French

Advanced Attribute Selectors

5. [attribute^="value"]

Selects elements with attribute starting with value

[href^="https"]
Secure link Regular link

6. [attribute$="value"]

Selects elements with attribute ending with value

[src$=".jpg"]

7. [attribute*="value"]

Selects elements with attribute containing value

[alt*="cat"]

8. [attribute*="value" i]

Case-insensitive matching

[class*="button" i]
